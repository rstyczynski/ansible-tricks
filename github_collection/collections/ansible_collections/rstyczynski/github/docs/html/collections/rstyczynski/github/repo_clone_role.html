rstyczynski.github.repo_clone role – Clones GitHub repository or updates existing clone with branch sync

Note

This role is part of the rstyczynski.github collection (version 1.0.0).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it use: ansible-galaxy collection install rstyczynski.github.

To use it in a playbook, specify: rstyczynski.github.repo_clone.
