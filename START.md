# Start

Work on this directory / project as main goal, by using agentic instructions and files from RUPStrikesBack project.

What have to be done is defined in `backlog.md` with iterations defined in `plan.md`.

How to proceed - all instructions, rules, etc. are taken from RUPStrikesBack. Add RUPStrikesBack `https://github.com/rstyczynski/RUPStrikesBack.git` as a submodule and run `@RUPStrikesBack/.claude/commands/rup-manager.md`.

## To keep RUPStrikesBack up to date

```
git clone --recursive
```

```
git submodule update --remote
```
